Welcome to MixPack!

A mixed TypeScript/JavaScript Webpack boilerplate with Express.

Features:

  • Transpile Typescript and Next-Gen JavaScript
  • Run TypeScript and JavaScript test files using Jest.
  • Supports multi-parser linting for TypeScript and JavaScript using Eslint
  • Integrates the Express + Webpack (expack) boilerplate with hot-reloading.
  • Transpile Sass and Nex-Gen CSS.
  • Performance optimized webpack configurations.
  • Code Splitting example files included!
